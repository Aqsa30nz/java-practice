
# Java Practice Programs

This repository contains my Java practice programs created while revising core Java concepts.

## Structure
- basics/ : Fundamental Java programs (loops, conditionals, logic building)
- games/  : Small interactive programs and logic-based games

## Topics Covered
- Input / Output
- Conditional statements
- Loops
- Functions
- Number problems
- Pattern printing
- Simple games

## Purpose
This repository is part of my learning journey to strengthen Java fundamentals
before moving to Data Structures, Backend development, and Open Source (GSoC).

## How to Run
1. Compile the program  
   javac FileName.java
2. Run  
   java FileName
